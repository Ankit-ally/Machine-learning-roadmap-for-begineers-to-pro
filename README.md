## 📌 Project Overview

This project focuses on **analyzing and predicting pricing trends** in the **Indian consumer electronics market**, specifically for niche audio products.

It covers the complete **Data Science lifecycle**, from data exploration to model deployment concepts, making it a **comprehensive end-to-end project**.

* Understand pricing patterns of audio products
* Perform deep exploratory data analysis (EDA)
* Handle real-world data issues (missing values, outliers, skewness)
* Build and compare multiple machine learning models
* Evaluate model performance using regression & classification metrics
